# 💈 DannyWacho Barbershop

A professional barbershop website built with HTML, CSS, and JavaScript. Featuring an online appointment booking system, service gallery, and admin dashboard.

## ✨ Features

- **Responsive Design** - Mobile-friendly interface that works on all devices
- **Online Booking System** - Customers can book appointments directly
- **Service Gallery** - Showcase of barbershop styles and services
- **Admin Dashboard** - View and manage all bookings
- **AI Chatbot** - Automated responses to customer inquiries
- **Google Maps Integration** - Show your exact location
- **Contact Information** - Phone, email, and hours of operation

## 📋 Services Offered

- Fade Haircuts
- Skin Fades
- Beard Trimming & Styling
- Hair Wash & Treatment
- Kids Haircuts
- VIP Package (Haircut + Beard Trim + Facial Treatment)

## 🌐 Sections

- **Home** - Hero section with introduction and call-to-action
- **About Us** - Information about the barbershop and team
- **Services** - Detailed service cards
- **Gallery** - Photo gallery of work
- **Booking** - Online appointment form
- **Contact** - Location, phone, email, and hours

## 🚀 Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Navigate through the website using the menu
4. Book appointments and manage bookings via the admin section

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactivity and local storage
- **Google Maps API** - Location embedding

## 💾 Data Storage

Bookings are stored in the browser's local storage. Data persists until the browser cache is cleared.
